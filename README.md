# 10closest
iOS App showing the 10 closest locations using GoogleAPI


Task
Objetivo
Um app simples, desenvolvido em linguagem nativa (Kotlin ou Swift), que tem como objetivo mostrar os lugares mais próximos ao usuário, dado um termo inserido. Exemplo: O usuário digita farmácia, seu app deve indicar no mapa as 10 farmácias mais próximas.

Ferramentas
Você deve usar a API do Google Maps para trazer os lugares relacionados à busca
Implemente testes em seu app, quanto mais melhor ;)
Seu app deve ser colocado no Github para avaliação

Notas e dicas
Capricho é f-u-n-d-a-m-e-n-t-a-l. Desenvolva algo com a qualidade que você colocaria em produção!
Cumprindo o objetivo principal, você está livre para criar e adicionar features que achar que fazem sentido
Lembre-se de não versionar senhas, chaves de API, etc, em seu código. Deixe claro em qual parte devemos inserir as credenciais de autenticação para API do Google Maps, que seguiremos de lá ;)
Restrições

O código deve ser nativo, ou seja, sem nenhuma ferramenta para cross platform.
